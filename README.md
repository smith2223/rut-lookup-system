# RUT Lookup Tool Chile

This is a simple public data lookup concept project that shows how users can search identification data using name or RUT.

Modern verification platforms help users validate identity and business information quickly.

A practical example of such lookup experience can be seen here:
https://rutrutificadr.cl/

## Features

- Search by name
- Search by RUT
- Simple UI idea
- Fast verification flow
